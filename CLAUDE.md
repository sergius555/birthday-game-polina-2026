# Космический квест ко дню рождения Полины (9 лет)

Тема — «Тайна третьей планеты» (Кир Булычёв). Полине дают iPad как «бортовой компьютер»
корабля «Космический Единорог». Набор HTML-страниц во вкладках. Всё на русском (с «ё»).
Тон: тепло, дружелюбно, прощающе (ошибка ≠ проигрыш квеста).

Стиль: тёмный космос, звёзды. Шрифты: Playfair Display (заголовки), Oswald (HUD),
PT Serif (текст), Share Tech Mono (техно). Цвета: фон #05060e, циан #5fd0e6, янтарь #e6b552, текст #e7e3d5.

## Хостинг / деплой

  (SSH-ключа нет). ⚠️ Присланный в чат токен `ghp_...` был засвечен — надо отозвать/сменить.

## Уроки iPad / WebKit (важно!)
- Картинки должны совпадать по реальному формату с расширением: `alisa`/`robot` были PNG под `.jpg`
  → Safari не показывал → переименованы в `alisa.png`, `robot.png`.
- Видео (`<video>`) на iOS требует `playsinline muted loop autoplay` (иначе не играет / кнопки).
- Имена файлов латиницей (кириллица в URL рискованна): svyaz/nav/ship/log.
- Звук нельзя играть до первого касания → экран авторизации (отпечаток) разблокирует аудио.

## Прогрессия (состояние = `level`)
1. **Земля** — звонок/письмо Алисы → Газета (намёк на Венеру) → ввод «Венера».
2. **Раннер** (полёт к Венере) → level 3.
3. **Венера** — письмо Механика Салатового → ввод «Блук» → авария реактора → починка в «Корабле» → прилёт (level 4).
4. **Блук** — письма Папы и Ушанов (звонок) → пазл-QR → ввод «Шелезяка» → **shooter** (продержаться 30 сек) → level 5.
5. **Шелезяка** — после shooter навигация ломается (`nav:broken`): на вкладке Навигация — треснувший
   экран с ошибкой → в «Корабль» → мини-игра «луч через зеркала» → `nav:fixed`. Письмо Фиолетового →
   письмо Робота (ЛОВУШКА) → «Диск Цезаря» (сдвиг 7, `ЯМЯПФМЧЪ`→`ШЕШИНЕРУ`) → ввод «Шешинеру» → level 6.
6. **Шешинеру** (планета Хранителей Времени, золотой шар с вращающимся кольцом-часами) — письмо Крыса
   (глумится, ушёл в прошлое спасать Весельчака У; аватар krys.jpg) → выход → звонок **Шешинерийцев**
   (Хранители Времени, голос audio_sheshinery.mp3) → кнопка «Показания Хроно-компаса»: 4 показания-истории
   (примеры на ×/+/− с подвохом на порядок действий) → дата **27.11.2178**. Ввод даты в поле Навигации
   (27112178 или с точками) → **анимация переноса во времени** → прибытие в прошлое (level 7): кольцо-часы
   крутится в обратную сторону, метка «Прошлое · 27.11.2178». Финал (поимка Крыса) — TODO.

Мини-игры: раннер (nav, 10 сек, прыжок/не-прыжок), починка-переливание + починка навигации «луч через
зеркала» (ship), пазл-QR + shooter (nav/svyaz), диск Цезаря (svyaz).
Планеты в Навигации отличаются видом/эмодзи (🌍🟡🔵⚙️🛸). Цель каждого уровня — только текущая планета.

## Персистентность
Источник правды — ВЕРХНЕЕ окно index.html: `localStorage['edinorog_state']` + дубль `window.name`
(переживает F5 даже когда Safari блокирует storage). Вкладки-iframe не хранят сами — спрашивают/
обновляют через postMessage (`req:state`/`set:...`). Объект:
`{ level, callSeen, mailRead:[ids], mailLater, ship, scraps, decoded }`.
Каждый шаг фиксируется при вводе планеты / прохождении и не повторяется.

## Письма (svyaz.html)
Стартовые: Космопочта, Галакто-Маркет(спам), Мама и Папа, Алиса(звонок, срочно).
